# CloudCast

CloudCast is a real-time weather forecasting web application that provides accurate weather updates for any location.  
It displays temperature, humidity, wind speed, climate visuals, and AQI index using the OpenWeatherMap API.

**Live Demo:** [View Website](https://shyamkumarnagilla.github.io/CloudCast/)

---

## Features

- Real-time weather data using OpenWeatherMap API  
- Displays temperature, humidity, wind speed, and AQI index  
- Responsive and visually appealing user interface  
- Smooth transitions with an organized layout  
- Fast and lightweight design for better performance  

---

## Technologies Used

- HTML  
- CSS  
- JavaScript  
- OpenWeatherMap API
